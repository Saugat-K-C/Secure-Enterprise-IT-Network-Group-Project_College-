# Secure-Enterprise-IT-Network-Project_College
Design, Installation and Configure Network Servers for an enterprise network
# Project Scenario
Scenario  
In this project. a company wants us to implement and install following windows server services and these were our main task:
Windows web server (IIS), called XYZ Tech Solutions, to host a simple company website. 
Backup server, called Back_SiliconValley, 
DHCP server, called DHCP_SiliconValley,
Roaming profiles for the users. 
Deploy Active directory domain services for managing groups, users and resources within a network. 
Deploy Remote Desktop Services. 
Implement Network Policy and Access Services 
Implement Group Policy to enforce a password policy. 
## My Role
We were a group of five members where I contributed as a,
**Lead Windows Server Administrator **  
- Installed and configured IIS, Active Directory, DHCP, RDS, and Roaming Profiles  
- Designed network topology and IP configuration  
- Implemented Network Policy, access controls, and Group Policies
 and 
- My team members contributed by preparing detailed project documentation, creating network diagrams, and producing demonstration videos for presentation.
